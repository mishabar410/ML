Проект Voice_cloning представляет собой попыку реализовать "аудио дипфейк", который должен подменять голос человека в аудиозаписи.

numpy_creating.py отвечает за создание numpy массивов из аудиофайлов
gan_realization.py это сама нейросеть. 
В реализации используется автоэнкодер и генеративно-состязательная нейросеть.
